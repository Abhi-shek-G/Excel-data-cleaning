# Excel-data-cleaning
This project demonstrates fundamental data cleaning and formatting techniques using Google Sheets / Microsoft Excel on a real-world dataset. The goal is to transform a raw, messy dataset into a clean, analysis-ready format, following industry-style data preparation practices.

Dataset
Source: Kaggle (Netflix Movies and TV Shows)

Data Cleaning Steps Performed:

Imported raw CSV data with proper headers and delimiters
Created separate sheets for Raw_Data, Backup_Data, and Cleaned_Data
Identified and handled missing values using filters and conditional formatting
Removed duplicate records after creating a backup copy
Standardized text fields using functions like TRIM, PROPER, and UPPER
Corrected data formats for dates, numbers, and categorical values
Maintained raw and cleaned data separation (professional pipeline practice)
Added a Data_Quality_Notes column to document data issues and cleaning decisions
Exported final cleaned dataset in XLSX and CSV formats
